# [4조] StudySync 개발계획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 프로젝트명 | StudySync |
| 주제 | MediaPipe 기반 자세 감지 + AI 재학습 + 집중도 관리 개인 공부 스케줄러 |
| 개발 목적 | 웹캠으로 사용자의 자세와 졸음, 자리이탈 상태를 분석하여 공부 집중도를 실시간 판단하고 휴식 알림, 목표 관리, 통계 리포트를 제공한다. |
| 개발 기간 | 10일 (Phase 1: 7일 / Phase 2: 3일) |
| 팀 구성 | 정재훈(AI서버) / 김혜윤(메인서버) / 김찬우(클라이언트) / 정태현(PM·통합) |
| 개발 환경 | Python 3.11, PyQt5, OpenCV, MediaPipe, FastAPI, PyTorch, MySQL |
| 서버 구성 | AI 학습서버 + 메인 운용서버 |

## 2. 핵심 기능

### 2.1 자세 감지 및 집중도 판단

- MediaPipe Pose 기반 33개 랜드마크 실시간 추출
- 목 각도, 어깨 기울기, EAR, 자리이탈 여부 분석
- 클라이언트는 30fps로 웹캠 화면 렌더링
- AI 서버에는 5fps만 전송하여 분석 부하 분리
- Phase 1에서는 규칙 기반 집중도 점수 계산
- Phase 2에서는 PyTorch 재학습 모델로 집중/딴짓/졸음 3분류

### 2.2 회원 및 목표 관리

- 회원가입 및 로그인
- JWT 기반 인증 처리
- 하루 공부 목표 시간 설정
- 휴식 주기와 휴식 시간 설정
- 90분 이상 연속 집중 시 과집중 방지 알림 제공

### 2.3 통계 및 리포트

- 오늘 집중 시간과 휴식 시간 분리 표시
- 평균 집중도 및 경고 횟수 제공
- 시간대별 집중도 그래프 제공
- 주간 공부량 트렌드 제공
- 사용자의 황금 집중 시간대 분석
