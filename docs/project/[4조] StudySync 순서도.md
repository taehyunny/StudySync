## 1. 전체 흐름

```mermaid
flowchart TD
    A["사용자: 앱 실행"] --> B["로그인"]
    B --> C["목표 설정"]
    C --> D["공부 세션 시작"]
    D --> E["클라이언트: 웹캠 30fps 렌더링"]
    E --> F["클라이언트: 5fps 프레임 추출"]
    F --> G["AI서버: MediaPipe 분석"]
    G --> H["AI서버: 집중도·자세 결과 반환"]
    H --> I["클라이언트: 오버레이·게이지·알림 표시"]
    I --> J["메인서버: 5초마다 로그 저장"]
    J --> K["메인서버: 통계 집계"]
    K --> L["클라이언트: 대시보드 표시"]
    L --> M["세션 종료"]
```
