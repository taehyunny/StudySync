# [4조] StudySync 디자인목업

## 1. 화면 구성

| 화면 | 구성 요소 | 담당 |
|---|---|---|
| 로그인 | 이메일·비밀번호 입력, 로그인 버튼, 회원가입 링크 | 정태현 |
| 메인 화면 | 웹캠 뷰, 랜드마크 오버레이, 집중도 게이지, 상태 뱃지, 목 각도 수치, 공부 타이머 | 정태현 |
| 알림 팝업 | 자세 경고, 휴식 권장, 과집중 강제 알림 | 정태현 |
| 목표 설정 | 하루 목표 시간, 휴식 주기, 휴식 시간, 달성률 Progress Bar | 정태현 |
| 통계 대시보드 | 오늘 집중 시간, 휴식 시간, 시간대별 Bar Chart, 주간 트렌드, 경고 횟수 | 정태현 |
| 라벨링 화면 | 집중/딴짓/졸음 버튼, 라벨별 수집 현황 | 정태현 |

## 2. 주요 화면 설명

### 2.1 로그인 화면

- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 회원가입 화면 이동 링크
- JWT 저장 후 자동 로그인 처리

### 2.2 메인 화면

- 중앙에 웹캠 화면 표시
- MediaPipe 랜드마크 오버레이 표시
- 집중도 점수를 Progress Bar로 표시
- 현재 상태를 집중/주의/경고/딴짓/졸음 뱃지로 표시
- 공부 타이머와 오늘 목표 달성률 표시

### 2.3 알림 팝업

- 자세 경고: 목 각도 또는 어깨 기울기가 기준을 초과할 때 표시
- 휴식 권장: 설정한 휴식 주기에 도달했을 때 표시
- 과집중 강제 알림: 90분 이상 연속 집중 시 표시

### 2.4 목표 설정 화면

- 하루 목표 공부 시간 입력
- 휴식 주기 입력
- 휴식 시간 입력
- 목표 달성률 Progress Bar 표시

### 2.5 통계 대시보드

- 오늘 집중 시간
- 오늘 휴식 시간
- 평균 집중도
- 경고 횟수
- 시간대별 집중도 그래프
- 주간 공부량 트렌드

### 2.6 라벨링 화면

- 집중, 딴짓, 졸음 라벨 버튼
- 현재 수집된 라벨별 데이터 개수 표시
- Phase 2 재학습 데이터 수집에 사용
