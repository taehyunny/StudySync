# 4조 StudySync - 정태현(PM·통합)

## 1. 담당 범위

정태현은 StudySync의 PM 및 통합을 담당한다. 전체 요구사항을 정리하고 개발 일정을 관리하며, AI서버·메인서버·클라이언트가 정상적으로 연동되는지 점검한다. 최종 발표 자료와 문서화도 함께 담당한다.

## 2. 주요 업무

| 업무 | 설명 |
|---|---|
| 요구사항 정리 | 기능 범위, 우선순위, 담당자별 작업 정리 |
| 일정 관리 | Phase 1, Phase 2 개발 일정 관리 |
| API 명세 검토 | 서버 간 요청/응답 형식 일치 여부 확인 |
| 통합 테스트 | 클라이언트-AI서버-메인서버 전체 흐름 검증 |
| QA | 버그 목록 작성, 수정 여부 확인 |
| 문서화 | 개발계획서, 요구사항 분석서, 통신 프로토콜 정리 |
| 발표 준비 | 발표 자료, 시연 흐름, 데모 체크리스트 작성 |

## 3. 통합 테스트 항목

| ID | 테스트 항목 | 완료 기준 |
|---|---|---|
| QA-01 | 로그인 후 메인 화면 진입 | JWT 저장 및 인증 API 호출 성공 |
| QA-02 | 웹캠 프레임 AI서버 전송 | 5fps 분석 요청 정상 처리 |
| QA-03 | 집중도 결과 표시 | 게이지와 상태 뱃지가 AI 응답과 일치 |
| QA-04 | 로그 저장 | 5초 단위 집중도/자세 로그 DB 저장 |
| QA-05 | 세션 종료 | 집중 시간과 평균 집중도 계산 |
| QA-06 | 통계 대시보드 | 오늘 통계와 주간 리포트 표시 |
| QA-07 | 라벨링 데이터 수집 | 집중/딴짓/졸음 데이터 저장 |
| QA-08 | 재학습 모델 통합 | 모델 정확도 검증 및 API 응답 확인 |

## 4. 산출물

- 전체 개발 일정표
- 요구사항 분석서
- API 명세서
- 통합 테스트 체크리스트
- 최종 발표 자료
- 데모 시나리오
