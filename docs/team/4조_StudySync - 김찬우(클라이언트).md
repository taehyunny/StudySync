# 4조 StudySync - 김찬우(클라이언트)

## 1. 담당 범위

김찬우는 StudySync의 클라이언트를 담당한다. MFC 기반 UI를 구현하고 OpenCV 웹캠 화면을 30fps로 렌더링하며, 샘플링 프레임을 ZMQ로 AI서버에 전송한다. AI 분석 결과를 화면에 오버레이하고 메인서버에 집중도와 자세 로그를 저장한다.

## 2. 주요 기능

| 기능 | 설명 |
|---|---|
| 로그인/회원가입 UI | 이메일, 비밀번호 입력 및 JWT 저장 |
| 웹캠 렌더링 | OpenCV 프레임을 MFC/Direct2D 화면에 30fps 표시 |
| AI 서버 전송 | 200ms마다 프레임을 base64 인코딩 후 전송 |
| 자세 오버레이 | 랜드마크, 목 각도, 교정 표시 렌더링 |
| 집중도 게이지 | 0~100점 Progress Bar 및 상태 뱃지 표시 |
| 알림 팝업 | 자세 경고, 휴식 권장, 과집중 알림 |
| 목표 설정 UI | 목표 시간, 휴식 주기, 휴식 시간 설정 |
| 통계 대시보드 | 오늘 통계, 시간대별 그래프, 주간 트렌드 표시 |
| 라벨링 버튼 | 집중/딴짓/졸음 데이터 수집 버튼 |

## 3. 화면 목록

| 화면 | 설명 |
|---|---|
| 로그인 화면 | 사용자 인증 및 회원가입 진입 |
| 메인 화면 | 웹캠, 집중도, 상태, 공부 타이머 표시 |
| 목표 설정 화면 | 공부 목표와 휴식 설정 |
| 통계 대시보드 | 공부량 및 집중도 통계 표시 |
| 라벨링 화면 | Phase 2 재학습 데이터 수집 |

## 4. 서버 연동

| 대상 | 연동 내용 |
|---|---|
| AI서버 | `/analyze/frame`, `/baseline/capture`, `/traindata` |
| 메인서버 | `/auth/login`, `/goal`, `/session/start`, `/focus`, `/posture`, `/stats/*` |
