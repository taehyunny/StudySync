# 4조 StudySync - 정재훈(AI서버)

## 1. 담당 범위

정재훈은 StudySync의 AI 학습서버를 담당한다. 클라이언트에서 전달받은 웹캠 프레임을 분석하여 자세, 졸음, 자리이탈 여부를 판단하고 집중도 점수와 교정 가이드를 반환한다. Phase 2에서는 사용자 라벨링 데이터를 기반으로 PyTorch 재학습 모델을 구현한다.

## 2. 주요 기능

| 기능 | 설명 |
|---|---|
| 랜드마크 추출 | MediaPipe Pose 기반 33개 신체 좌표 추출 |
| 목 각도 계산 | 귀-어깨-수직선 기준 각도 계산 |
| 졸음 감지 | Face Mesh 기반 EAR 값 계산 |
| 자리이탈 감지 | 얼굴 미탐지 지속 시간 기준 이탈 판정 |
| 집중도 계산 | 자세, 눈, 자리, 정면 상태를 조합해 0~100점 산출 |
| 기준 자세 비교 | 세션 시작 시 기준 자세를 저장하고 이후 변화량 계산 |
| 재학습 데이터 수집 | 집중/딴짓/졸음 라벨별 랜드마크 JSON 저장 |
| PyTorch 모델 학습 | MLP 기반 3분류 모델 학습 및 정확도 검증 |

## 3. API

| 메서드 | 엔드포인트 | 설명 |
|---|---|---|
| POST | `/analyze/frame` | 프레임 분석 후 집중도·상태·각도·가이드 반환 |
| POST | `/baseline/capture` | 기준 자세 저장 |
| POST | `/traindata` | 라벨링 학습 데이터 저장 |
| GET | `/traindata/count` | 라벨별 데이터 수 조회 |
| POST | `/model/train` | 재학습 모델 학습 실행 |
