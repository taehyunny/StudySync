# Integration

담당: 정태현

## 책임 범위

- AI 서버, 메인서버, 클라이언트 API 명세 일치 여부 확인
- 통합 테스트 체크리스트 관리
- QA 결과와 버그 수정 상태 관리
- 발표 자료, 데모 시나리오, 최종 산출물 정리

## 권장 구조

```text
integration/
├─ qa/                 # 통합 테스트 체크리스트, 버그 리포트
└─ demo/               # 발표 시연 흐름, 데모 데이터
```

## 기본 통합 테스트

| ID | 항목 | 완료 기준 |
|---|---|---|
| QA-01 | 로그인 후 메인 화면 진입 | JWT 저장 및 인증 API 호출 성공 |
| QA-02 | 웹캠 프레임 AI 서버 전송 | 5fps 분석 요청 정상 처리 |
| QA-03 | 집중도 결과 표시 | 게이지와 상태 뱃지가 AI 응답과 일치 |
| QA-04 | 로그 저장 | 집중도/자세 로그가 DB에 저장 |
| QA-05 | 세션 종료 | 집중 시간과 평균 집중도 계산 |
| QA-06 | 통계 대시보드 | 오늘 통계와 주간 리포트 표시 |
| QA-07 | 라벨링 데이터 수집 | 집중/딴짓/졸음 데이터 저장 |
| QA-08 | 재학습 모델 통합 | 모델 정확도 검증 및 API 응답 확인 |
